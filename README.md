# CUET Online Stationery

**CUET Online Stationery** is a web platform designed for CUET students to browse, order, and receive stationery items such as pens, notebooks, lab materials, and art supplies directly at campus halls or designated pickup points.

## Features
- Browse and search stationery items by category, brand, or price  
- Add products to cart and checkout with secure payment (bKash or Cash on Delivery)  
- Track order status in real-time  
- Admin dashboard for managing inventory and orders  
- AI-driven product recommendations  
- Discounts and feedback system  

## Tech Stack
- **Frontend & Backend:** Next.js (React framework)  
- **Database:** JSON / Local storage / Optional backend integration  
- **Payment Integration:** SSLCommerz (sandbox)  

## Getting Started
1. Clone this repository:  
   ```bash
   git clone https://github.com/yourusername/cuet-stationery.git
   ```  
2. Install dependencies and run the app:  
   ```bash
   npm install
   npm run dev
   ```  
3. Open [http://localhost:3000](http://localhost:3000) in your browser.
